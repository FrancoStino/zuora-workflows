# Zuora Workflow Manager

<TechnologiesBadge badges={['Laravel 12', 'Filament 4.2', 'PHP 8.4', 'MariaDB 11.4']} />

<Alert type="warning">
  This project requires <strong>Laravel 12</strong>, <strong>PHP 8.4+</strong>, and proper <strong>Zuora API credentials</strong> to function correctly.
</Alert>

Zuora Workflow Manager is a powerful web application for synchronizing, viewing, and managing Zuora workflows directly from your Filament admin dashboard. Built with modern Laravel architecture featuring automated sync jobs, real-time dashboards, and comprehensive workflow management capabilities.
