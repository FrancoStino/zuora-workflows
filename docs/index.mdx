# Zuora Workflow Manager

Zuora Workflow Manager is a powerful web application for synchronizing, viewing, and managing Zuora workflows directly from your Filament admin dashboard. Built with modern Laravel architecture featuring automated sync jobs, real-time dashboards, and comprehensive workflow management capabilities.

<div className="warning">
  <p><strong>Note:</strong> This project requires Laravel 12, PHP 8.4+, and proper Zuora API credentials to function correctly.</p>
</div>

## Technology Stack

### Backend
- **Framework**: Laravel 12.x
- **PHP**: 8.4+
- **Database**: MariaDB 11.4
- **Queue**: Database/Redis queue driver
- **Authentication**: Filament + Filament Socialite (Google OAuth)

### Frontend
- **UI Framework**: Filament 4.2
- **Styling**: Tailwind CSS 4
- **Build Tool**: Vite 7
- **Graph Visualization**: JointJS with Dagre layout
- **Reactivity**: Alpine.js

### Key Dependencies
- `bezhansalleh/filament-shield` - Role-based access control
- `dutchcodingcompany/filament-socialite` - OAuth integration
- `moox/jobs` - Job monitoring and management
- `spatie/laravel-settings` - Application settings management
- `filament/spatie-laravel-settings-plugin` - Filament settings integration

## Use Cases

### For Business Users

- **Monitor Workflow Status**: View all Zuora workflows in one place
- **Track Task Progress**: See individual tasks within workflows
- **Visualize Workflows**: Interactive graphical representation of workflow flows
- **Export Data**: Download workflow definitions in JSON format

### For Developers

- **API Integration**: Seamless Zuora REST API integration with OAuth 2.0
- **Queue Processing**: Background job processing for non-blocking operations
- **Debug Tools**: Job monitoring with Moox Jobs integration
- **Extensible**: Easy to add custom features via Filament resources

### For Administrators

- **Multi-tenant**: Manage multiple customers with separate Zuora credentials
- **Access Control**: Role-based permissions via Filament Shield
- **Settings Management**: Centralized configuration with encrypted secrets
- **Automated Syncs**: Schedule automatic workflow synchronization
