## Caratteristiche principali

### Sincronizzazione automatica
Sincronizzazione configurabile (ogni ora di default) con pulsanti di sync manuale per aggiornamenti immediati.

### Dashboard Filament
Interfaccia ricca per visualizzare workflow e task con ricerca, filtri e ordinamento avanzati.

### Background Jobs
Elaborazione basata su code con logica di retry (3 tentativi, backoff esponenziale di 60s).

### Gestione Task
Estrazione automatica dei task dal JSON dei workflow con visualizzazione dettagliata e filtri avanzati.

### Visualizzazione Grafica
Rappresentazione grafica interattiva dei workflow con @joint/layout-directed-graph.
