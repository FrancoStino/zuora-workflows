# Features

Zuora Workflow Manager provides comprehensive tools for managing Zuora workflows, tasks, and users. This guide details all available features.

## Workflow Management

### Automatic Synchronization

Workflows can be synchronized automatically from Zuora API:

<FeatureSection
  title="Sync Triggers"
  items={[
    {
      icon: 'ðŸ‘†',
      title: 'Manual Sync',
      description: 'Click "Sync Workflows" button on customer record'
    },
    {
      icon: 'â°',
      title: 'Scheduled Sync',
      description: 'Automatic hourly sync (configurable via routes/console.php)'
    },
    {
      icon: 'ðŸ’»',
      title: 'CLI Command',
      description: 'php artisan app:sync-workflows --customer="Name"'
    }
  ]}
/>

<FlowDiagram type="sync-triggers" title="Sync Process Flow" />

<Alert type="info">
  <strong>Sync Process Details:</strong>
  <ol className="list-decimal list-inside mt-2 space-y-1">
    <li>Retrieves OAuth access token (cached for 1 hour)</li>
    <li>Fetches workflows from Zuora API (paginated, 50 per page)</li>
    <li>Downloads complete workflow JSON for each workflow</li>
    <li>Saves/updates workflow records in database</li>
    <li>Automatically extracts tasks from workflow JSON</li>
    <li>Removes workflows no longer present in Zuora</li>
  </ol>
</Alert>

<Alert type="warning">
  <strong>Retry Logic:</strong> Failed sync jobs are automatically retried up to 3 times with 60-second exponential backoff. Failed jobs are logged and visible in Moox Jobs panel.
</Alert>
